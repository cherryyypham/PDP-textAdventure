# Monty Python Text Adventure Game

A text-based adventure game based on "Monty Python and the Holy Grail" implementing MVC design pattern in Java.

## Project Structure

```
游닍 src
較럭Main.java                # Driver class      
較럭Controller.java          # Controller - manages game flow
較럭Model.java               # Model - stores game state
較럭View.java                # View - handles output display
較럭InputParser.java         # Custom text parser
較럭ㅁctionProcessor.java     # Action coordinator
較럭ㅁctionResult.java        # Result encapsulation
較럭DennisActions.java       # Dennis scene handler
較럭BlackKnightActions.java  # Black Knight scene handler
較럭FrenchCastleActions.java # French Castle scene handler
較덕KnightsNiActions.java    # Knights Ni scene handler
```

## Running the Program

1. Compile all Java files:
   ```bash
   javac *.java
   ```

2. Run the game:
   ```bash
   java Main
   ```

3. Play using verb-noun commands:
   ```
   > call peasant
   > ask castle
   > introduce
   > fight knight
   > run
   ```
